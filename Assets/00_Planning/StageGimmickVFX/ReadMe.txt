리소스 활용

Dust: 불안정한별 기믹 시작 시 맵 전체에 등장.
      각각 크기와 재생 속도가 다른 먼지 리소스들이 맵 전체에 등장
      이펙트와 캐릭터의 상호작용은 존재하지 않음

Fog: 안개 구름별 기믹 시작 시 맵 끝 테두리에서부터 등장

GreenGas: 초록가스 별 기믹 시작 시 랜덤 좌표값을 기준으로 스테이지에 가스 등장
          6개의 가스 이펙트가 한 번에 등장
          6개로 묶어서 하나의 프리펩으로 설정

Lightning: 낙뢰의별 기믹 시작 시 붉은 마커에 등장

Meteor: 분출하는 별 기믹 시작 시 붉은 마커에 등장
        메테오와 폭발 리소스를 함께 묶어 저장해두었음
        메테오가 내려오다가 지면에 닿을 때 폭발 이펙트 발생

Wind: 질풍의 별 기믹 시작시 맵 전체에 등장
      기믹 작용 방향에 맞게 바람 효과 진행 방향 설정
      리소스의 길이와 빈도 수는 적절하나 두께가 의도보다 두껍게 설정됨(수정 바람)
